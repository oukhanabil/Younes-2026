# SGA - Système de Gestion des Agents CleanCo

Application web de gestion des agents de la société CleanCo Service.

## 📋 Fonctionnalités

- **👥 Gestion des Agents** : Ajout, modification, suppression, import/export
- **📅 Gestion du Planning** : Planning mensuel, par groupe, par agent, échange de shifts
- **📊 Statistiques** : Globales, par agent, par groupe, classement
- **🏖️ Congés & Absences** : Gestion des congés, maladies, absences
- **🚨 Codes Panique** : Gestion des codes d'urgence
- **📻 Gestion Radios** : Inventaire, attribution, suivi
- **👔 Habillement** : Gestion des tenues et équipements
- **⚠️ Avertissements** : Suivi disciplinaire

## 🚀 Installation

1. Téléchargez tous les fichiers dans un dossier
2. Ouvrez `index.html` dans un navigateur moderne
3. L'application fonctionne entièrement en local (localStorage)
